#  自动添加index.vue index.stylus
### 右键文件夹点击create-vue-component-or-page。自动生成对应index.vue.index.stylus.vue文件自动引入index.stylus。vue中namespace自动使用文件夹的名称，文件夹名称注意使用驼峰方式命名。
